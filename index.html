About the Project

The LHDR aims to aid search and rescue operations by detecting the presence of a live human autonomously. It uses non-contact temperature sensors and image processing algorithms to correctly identify and detect humans in the vicinity. The system can either be run autonomously or be controlled manually, in each case the operator uses a smartphone to control/monitor the bots. The accuracy of the bot can vary due to multiple factors and the accuracy of the bot is still being tested in different conditions.

Build

The bot uses BeagleBone Black to control the sensors and camera.
